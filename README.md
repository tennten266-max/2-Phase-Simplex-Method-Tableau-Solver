# 2-Phase Simplex Method Tableau Solver in C
(2段階単体法)

有理数（分数）構造体を用いて、線形計画問題の2段階単体法（辞書形式）をシミュレーションし、各ステップのタブローを描画するC言語プログラムです。

Blandの規則（最小添字規則）を実装しているため、巡回（無限ループ）を回避し、確実に最適解または非有界・実行不能の判定へ到達します。

## 特徴

- 分数による正確な計算: 浮動小数点数（double等）の丸め誤差が発生せず、すべての係数を 1/3 や -2/5 といった有理数形式で厳密に計算します。
- 2段階単体法に対応: 初期状態で基底解が実行不可能（定数項 b に負の値がある）な場合、自動的に人工変数 xa を導入して第1段階（補助問題）からスタートします。
- クリアなタブロー表示: 罫線（| や -+-）を用いて、目的関数行、定数項列、各制約条件の境界を視覚的に分かりやすく出力します。

---

## 使い方と入力形式

プログラムを起動すると、標準入力（端末への打ち込みやリダイレクト）から問題のデータを読み込みます。

### 入力形式のルール

1. 1行目: [制約式の数 m] [元の変数の数 n] をスペース区切りで入力します。
2. 2行目: 目的関数 Z の係数を n 個入力します。
3. 3行目以降: 各制約式の係数 n 個と、その式の定数項（右辺の値を符号反転させたもの、または辞書形式の定数項）を合わせた n+1 個の数値を m 行分入力します。

注意: 途中に余分な改行やスペースがあってもバグが起きないよう、個数指定による堅牢な読み込みを採用しています。

---

## 入力例と出力イメージ

### 例題（変数 2個、制約条件 3個）

最小化 z = -3*x1 - 2*x2
制約条件:
  2*x1 - 1*x2 <= -1
 -1*x1 + 2*x2 <= 4
 -1*x1 - 1*x2 <= -2

#### 入力データ (input.txt)
```text
3 2
-3 -2
2 -1 -1
-1 2 4
-1 -1 -2
```
